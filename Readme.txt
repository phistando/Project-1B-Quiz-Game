--------------------------- Pong Game ---------------------------

This is a simple Pong game created in javascript.


Game Introduction:

Pong is a two-dimensional sports game that simulates table tennis. The player controls an in-game paddle by moving it vertically across the left side of the screen, and can compete against either a computer-controlled opponent or another player controlling a second paddle on the opposing side. Players use the paddles to hit a ball back and forth.


Game Instructions:

Player 1 controls the paddle at the left hand side of the canvas.
Button ‘W’ for moving paddle up and button ’S’ for moving paddle down

Player 2 controls the paddle at the right hand side of the canvas.
Button ‘O’ for moving paddle up and button ‘L’ for moving paddle down
